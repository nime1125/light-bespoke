LIGHT-BESPOKE 사용방법
1 앱 설치
Android — Chrome 브라우저에서 이 페이지를 열고 앱 설치 시작 버튼을 탭하세요. 홈 화면에 아이콘이 생성됩니다.
iOS — Safari는 Web Bluetooth를 지원하지 않습니다. App Store에서 Bluefy 앱을 설치한 뒤, Bluefy 내에서 이 페이지 URL을 열어주세요.
PC (Chrome/Edge) — 별도 설치 없이 브라우저에서 바로 실행 가능합니다. Web Bluetooth 지원 필수.
2 BLE 페어링
1. 앱 상단의 PAIR 버튼을 탭합니다.
2. 조명 전원을 켜고 Bluetooth 모드를 활성화하세요.
3. 팝업 목록에서 연결할 기기를 선택합니다.
4. 연결되면 카드가 자동으로 생성되고 ● 녹색 상태 표시등이 켜집니다.
지원 브랜드:
GODOX SL / FL / MG / TL / KNOWLED 시리즈
APUTURE AL / Amaran / LS / Nova / MC 시리즈
NANLITE Pavo / Forza / FS / Halo / MixPad 시리즈
3 조명 제어
카드를 탭해서 펼치면 4개의 제어 탭이 나타납니다.

CCT 탭 — 색온도(K)와 밝기(%)를 슬라이더로 조정합니다.
HSI 탭 — RGB 지원 기기 전용. 컬러 휠을 터치하거나 Hue·Sat·Intensity 슬라이더로 색상을 설정합니다.
FX 탭 — 플래시·캔들·TV·폭풍 등 특수 효과를 선택하고 속도를 조절합니다.
LX 탭 — 번개 엔진. TAP / HOLD / AUTO 모드와 SLOW·MED·FAST·STORM 속도로 리듬 플래시를 연출합니다.
4 마스터 컨트롤
하단 MASTER 버튼을 탭하면 연결된 모든 조명을 동시에 제어할 수 있습니다.

· 마스터 디머 슬라이더로 전체 밝기를 0~100%로 조정
· 마스터 CCT로 전체 색온도를 일괄 변경
· ALL OFF 버튼으로 즉시 전체 소등
· SYNC ALL로 모든 기기에 현재 설정값을 재동기화
· BROADCAST FLASH 버튼으로 전체 기기에 동시 플래시
5 2D 맵 뷰
상단 우측 맵 아이콘을 탭하면 촬영 공간을 시각화한 2D 맵으로 전환됩니다.

· 각 조명 노드를 드래그하여 실제 위치에 배치할 수 있습니다.
· 노드를 탭하면 해당 기기의 컨트롤 시트가 아래에서 올라옵니다.
· 노드 색상이 현재 조명 색상을 실시간으로 반영합니다.
· 위치는 자동 저장되어 다음 실행 시 복원됩니다.
6 그룹 관리
상단 탭 바 우측의 ＋ 버튼을 탭하면 그룹을 생성할 수 있습니다.

· 여러 기기를 선택해 하나의 그룹으로 묶으세요.
· 그룹 탭을 선택하면 해당 기기들만 목록에 표시됩니다.
· ALL 탭은 전체, GODOX / APUTURE / NANLITE 탭은 브랜드별 필터링입니다.
7 자동 재연결 및 복원
연결이 끊어지면 자동으로 15회 재연결을 시도합니다 (지수 백오프 방식).

· 재연결 성공 시 이전 밝기·색온도·색상을 자동 복원합니다.
· 이전에 연결했던 기기 목록은 앱을 닫아도 로컬에 저장됩니다.
· 배터리 잔량과 신호 강도는 40초마다 자동 갱신됩니다.
8 기기추가 — 프로파일 보완
앱 상단 우측 기기추가 버튼을 탭하면 지원 기기 데이터베이스를 검색하거나 직접 추가할 수 있습니다.

검색으로 추가
· 브랜드(GODOX / APUTURE / NANLITE)를 선택하고 제품명을 입력하세요.
· 예: MS60C Amaran 300c Pavo Slim
· 결과 목록에서 [적용] 버튼을 탭하면 해당 기기의 BLE 프로파일이 즉시 저장됩니다.

직접 입력으로 추가
· 목록에 없는 신제품은 모델명 · BLE 기기명 prefix · 색온도 범위 · 지원 기능을 직접 입력해 저장할 수 있습니다.
· 저장된 프로파일은 이후 BLE 스캔 시 자동으로 인식됩니다.

DB 자동 업데이트
· 앱 실행 시 GitHub에서 최신 기기 DB를 자동으로 가져옵니다.
· 오프라인 환경에서는 마지막으로 저장된 캐시 버전을 사용합니다.
